> 25分集中して、サイコロでご褒美を決める！ゲーム感覚のポモドーロ Chrome 拡張機能

![License](https://img.shields.io/badge/License-MIT-yellow)

## ✨ 機能

- **ポモドーロタイマー** — 25分の作業 → 5分の休憩サイクルを自動管理
- **サイコロでご褒美決定** — 休憩開始時にサイコロを振って、6種類のご褒美からランダムに選択
- **当たり／ハズレの確率設定** — デフォルトで当たり67%・ハズレ33%（カスタマイズ可）
- **バックグラウンド動作** — ポップアップを閉じてもタイマーが継続
- **デスクトップ通知** — 作業終了・休憩終了をお知らせ
- **一時停止・再開** に対応
- **設定画面** から作業時間・休憩時間・ご褒美リストをカスタマイズ可能

## 📦 インストール方法

### 方法①：ZIPファイルから（かんたん）

1. [Releases](../../releases) ページから最新の `pomodoro-reward-timer.zip` をダウンロード
2. ZIPを解凍する（`pomodoro-ext` フォルダが出てきます）
3. Chrome で `chrome://extensions` を開く
4. 右上の **「デベロッパーモード」をオン** にする
5. **「パッケージ化されていない拡張機能を読み込む」** をクリック
6. 解凍した `pomodoro-ext` フォルダを選択

### 方法②：ソースから手動インストール

```bash
git clone https://github.com/YOUR_USERNAME/pomodoro-reward-timer.git
```

その後、上記の手順 3〜6 でフォルダを読み込む。

## 🖥️ 使い方

1. Chrome ツールバーの 🍅 アイコンをクリック
2. **「▶ 開始」** を押して25分タイマーをスタート
3. タイマーが終わると通知が届き、休憩モードに切り替わる
4. **「サイコロを振る！」** をクリックしてご褒美を確認
5. ご褒美（またはハズレ）を実行して休憩を楽しむ
6. 休憩タイマーが終わると、また作業モードに戻る

---

## ⚙️ カスタマイズ

ポップアップ右下の **「⚙️ 設定」** から変更できます。

- **作業時間**（デフォルト：25分）
- **休憩時間**（デフォルト：5分）
- **ご褒美リスト**（6項目すべて自由に編集可）
